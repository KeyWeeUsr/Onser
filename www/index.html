Place your files here.

By default the permissions will be changed this way:

  • www folder permissions to 0700 (user: rwx, group: ---, others: ---)
  • the contents of the www folder to 0644 (user: rw-, group: r--, others: r--).

Feel free to change the permissions via "inheritting" from the original Dockerfile with FROM keyweeusr/onser:<tag> instruction and modify the permissions like this:

FROM keyweeusr/onser:<tag>
USER root

# folder
RUN chmod 0123 /home/onser/www

# folder + files
RUN chmod -R 0123 /home/onser/www

USER onser
